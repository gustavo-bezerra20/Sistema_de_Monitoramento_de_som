# Hardware

Esta pasta contém os arquivos relacionados ao hardware do sistema de monitoramento de ruído.

## Componentes utilizados:
- ESP32-S3
- Microfone digital INMP441
- Fonte 3.3V

## Conexões:

INMP441 → ESP32-S3  
VCC → 3.3V  
GND → GND  
SCK → GPIO 6  
WS → GPIO 5  
SD → GPIO 4  
L/R → GND
